# AI Skills

This repository contains a collection of AI skills, organized into various categories. Each skill is designed to perform a specific task, such as code review in different domains.

## Directory Structure

- `docs/`: General documentation and guidelines.
- `skills/`: This directory contains all the AI skills.
  - `code-review/`: Skills related to foundation code review, as main source.
  - `code-reviewer-business-logic/`: Skills focused on reviewing business logic in code.
  - `code-reviewer-security/`: Skills specialized in identifying security vulnerabilities in code.
  - `code-reviewer-testing/`: Skills dedicated to reviewing testing strategies and code.
  - `refactoring/`: Skills related to systematic code improvement and refactoring.
  - `smart-commits/`: Skills for organizing and creating atomic git commits.
